Техническое задание:
Реализовать с использованием GAS игровую сцену. На сцене должны быть
1. Широкий коридор, в конце его дверь, за дверью комната
2. Перед дверью патрулирует бот-противник
3. Игрок стартует в начале коридора
4. Если игрок входит в зону аггро противника - тот атакует. Бежит к игроку и бьет его
5. У игрока есть определенное конечное кол-во жизни
6. По выходу жизни - смерть
7. По подходу к двери на ней появляется приглашение открыть
8. По выходу в комнату - бот перестает преследовать
9. У игрока есть абилка - спринт на шифт
10. Желательно использовать GAS, с++, state tree
